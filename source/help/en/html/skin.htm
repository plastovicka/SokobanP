Skins

If you want to change the skin, use commands in menu skin. Each skin is saved in one BMP file. You can cycle all skins from the same directory by commands Next or Previous. The name of the skin is shown in the window title and it disappears after few seconds.

You can make your own skin. You need some bitmap editor. (You can use Paintbrush from Windows accessories.)

The picture must consist of squares in this order:

  1. ground - an empty square
  2. object - a thing to push
  3. selected object - object hilighted by mouse click
  4. store - target square where a man should push the objects
  5. object on store - optional
  6. selected object on store - optional
  7. background - pattern for area around the level
  8. man - player
  9. wall

Squares for objects, selected objects and men are transparent bitmaps. Transparents colors are determined by the pixels in upper left corners of these squares.

You can create four squares for a man for directions left, right, up, down.

You can create five squares for a wall if you want adjacent walls to be connected. They must be placed in the picture in this order:

  1. without border - wall is surrounded by eight walls
  2. with border - wall is isolated, there are empty squares around this wall
  3. with corners -wall is connected with four other walls
  4. up and down border - wall adjoins with other walls at left and right side
  5. left and right border - wall adjoins with other walls at up and down side

There are eight types of skins:

  • 7 squares - 1x man, 1x wall
  • 9 squares - 1x man, 1x wall, 2x object on store
  • 10 squares - 4x man, 1x wall
  • 11 squares - 1x man, 5x wall
  • 12 squares - 4x man, 1x wall, 2x object on store
  • 13 squares - 1x man, 5x wall, 2x object on store
  • 14 squares - 4x man, 5x wall
  • 16 squares - 4x man, 5x wall, 2x object on store

The height of the picture must be same as height of one square. The width of the picture must be equal to the width of one square multiplied by number of squa res (which depends on skin type).

When you change the skin and save BMP file into skins folder, you can refresh the skin in Sokoban’s window.
