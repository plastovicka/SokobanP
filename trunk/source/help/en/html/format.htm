Files description

All files are saved in plain text format.

Symbols in levels:
# = wall
$ = object
. = target
@ = man
+ = man above target
* = object on target

Symbols in solutions:
0 = move left
1 = move right
2 = move up
3 = move down
4 = push left
5 = push right
6 = push up
7 = push down
C..Z = next symbol is repeated 3..26 times

Levels in the database “sokoban.dat” are stored without borders. Lines are separeted by !. Level, author and solutions are separated by ;.
