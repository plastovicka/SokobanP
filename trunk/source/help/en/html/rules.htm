Rules

You are a warehouse keeper. Your taks is to push all objects to target squares.

You cannot step over the objects. You can only move across squares that are not occupided.

You cannot push more than one object.

You cannot pull the objects.

Controls:

right mouse button - move to any place that can be reached without pushing any objects

left mouse button - select an object or push the selected object to mouse position

drag-and-drop - push an object

cursor keys - move one square left, right, up or down

numeric keypad - move one square in a direction, diagonal moves are counted as 2 moves

CTRL+cursor keys - move as far as possible without pushing any objects

SHIFT+cursor keys - move as far as possible, pushing any object in the way
